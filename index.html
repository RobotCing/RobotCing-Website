Robot Cing

Open source,easy to use robot for learning robotics created by Robot Cing Team.

Get Started

Educational tool

Robot Cing was designed as educational tool for those who want to learn robotics.

Easy to program

Robot Cing has 5 levels of programing it.

  • Programming using Scratch like enviroment
  • Programming using Cing Libraries
  • Programming using Arduino IDE enviroment
  • Programming using MicroPython
  • Programming using AVR language

    • Modular design

    Robot Cing has modular design.It has lot of sensors and modules protected against wrong insertation.

    Custom Arduino Software

    We developed custom board manager for Robot Cing so users can upload code form Arduino IDE. We have also created libraries which simplifies programming.

    Learning materials

    We have also created learning materials so users can learn quickly and effectively.
