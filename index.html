Robot Cing

Open source,easy to use robot for learning robotics created by Robot Cing Team.

Github Facebook Instagram

Educational tool

Robot Cing was designed to inspire people learning to write code to breath life and real personality into their own robot. He is loaded with tons of sensors allowing you to build thousands of robots for example line followers. User do not need any knowlage about electronics because Robot Cing can be progammed using libraries. Combine it with learning materials and you will get robot which can be programmed by anybody.

Features

Easy to program

Robot Cing has 3 levels of programming.

  • Programming using Cing Libraries
  • Programming using Arduino IDE enviroment
  • Programming using AVR language

    • Take your first steps into written code. Perfect for students who already know basics of programming, and even those completely new to coding.

    Modular design

    Robot Cing has modular design. It has got lot of sensors and modules protected against wrong insertation.

    Arduino board manager and Arduino Libraries

    We developed custom board manager for Robot Cing so users can upload code from Arduino IDE. We have also created libraries which simplifies programming. User can download our libraries with Arduino library manager.

    Learning materials

  • Teachers love the excellent free Robot Cing learning materials
  • No prior code experience necessary for students or teachers
  • Ideal for classroom instruction and workshops
  • Suitable for ages 8+ and younger if working with a grownup
